Tag-urile de formatare a textului:


1) p - paragraf

2) br - trecerea din rand nou

3) strong - textul ingrosat(bold)

4) em - text cursiv(italic)

5) pre - text preformatat

6) sub - index de jos

7) sup - index de sus

8) ins - subliniazatextul

9) del - text taiat

10) code - text in format de cod
